## Welcome to DayScribe

DayScribe is a local-first, privacy-preserving desktop activity tracker. It monitors your active windows and browser tabs, then generates AI-powered daily summaries so you never forget what you learned.

## Step 1: Install the Desktop App

Download the installer for your platform from the [Download page](/download). Run the installer and launch DayScribe. You'll see the dashboard at `http://localhost:9103`.

## Step 2: Install the Chrome Extension

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select the `Extension/` folder in the DayScribe directory
4. Pin the DayScribe extension icon

The extension starts reporting active URLs and tab titles to the local API.

## Step 3: Go About Your Day

Work as you normally would. DayScribe logs:
- Active window titles and app names every 2 seconds
- Browser tabs via the Chrome extension

## Step 4: Review Your First Digest

At the end of the day, open the DayScribe dashboard. Click **Daily Digest** to see:
- Top apps and time spent
- Browsed domains and articles
- AI-generated learning highlights

## Pro Tips

- **Tag your activity**: Use descriptive window titles to get better summaries.
- **Connect Ollama**: Install Ollama locally for free, private AI summarization.
- **Ask questions**: Use the AI Chat to query your data in natural language.

Happy tracking!
